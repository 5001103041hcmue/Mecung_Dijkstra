MÊ CUNG – TÌM ĐƯỜNG ĐI NGẮN NHẤT (DIJKSTRA)

Đây là đồ án mô phỏng mê cung trên nền tảng web và thực hiện tìm đường đi ngắn nhất giữa hai điểm bằng thuật toán Dijkstra.

LIÊN KẾT
- Demo (GitHub Pages): https://5001103041hcmue.github.io/Mecung_Dijkstra/
  
HƯỚNG DẪN CHẠY CHƯƠNG TRÌNH
1) Chạy trực tiếp (offline)
- Tải dự án về máy.
- Mở file index.html bằng trình duyệt Chrome hoặc Microsoft Edge.

2) Chạy bằng VS Code (khuyến nghị)
- Mở thư mục dự án bằng VS Code.
- Cài extension Live Server.
- Nhấn chuột phải vào index.html → chọn Open with Live Server.

CẤU TRÚC THƯ MỤC
- index.html: trang chính của ứng dụng.
- css/: các tệp định dạng giao diện.
- js/: các tệp xử lý giao diện và thuật toán.
- assets/: tài nguyên (hình ảnh, biểu tượng, âm thanh...).
- MazeMatix/: mô-đun liên quan đến biểu diễn mê cung/ma trận và xử lý đường đi (theo triển khai của nhóm).

GHI CHÚ
- Khuyến nghị sử dụng trình duyệt Chrome/Edge để đảm bảo hiển thị và hoạt động ổn định.
- Nếu không tồn tại đường đi hợp lệ, chương trình sẽ thông báo.
